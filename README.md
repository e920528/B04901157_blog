# B04901157_blog

## 環境要求
Ruby v2.4.1</br>
Rails 5.1</br>
Linux</br>
(由於環境設定有點麻煩，如果需要的話我可以直接跟助教約時間用我的電腦demo)

## How to Use
`git clone https://github.com/e920528/B04901157_blog.git`</br>
`cd B04901157_blog`</br>
`yarn install`</br>
`bundle install`</br>
`rails s`</br>
